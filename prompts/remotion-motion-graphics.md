# Remotion Motion Graphics Prompt

Act as a senior motion designer and React/Remotion engineer.

Create a Remotion composition for this project page.

## Goal

Create motion that improves the project storytelling and feels like a premium design studio case-study asset.

## Motion type

Choose one:
- hero loop
- project intro
- typography reveal
- image sequence
- process breakdown
- animated mockup
- social cut
- final payoff

## Rules

- Use existing brand colours.
- Use existing typography where possible.
- Do not distort logos.
- Keep motion subtle and premium.
- Avoid random effects.
- Use motion to reveal hierarchy or story.
- Export-ready for MP4/WebM.
- Include poster frame guidance.
- Respect reduced-motion usage on the website.

## Technical requirements

Use Remotion.
Create a clear composition with:
- width
- height
- fps
- durationInFrames
- clean props
- reusable constants
- no unnecessary dependencies

## Deliverables

1. Composition concept
2. Storyboard
3. Timing breakdown
4. Files to create
5. Remotion component code
6. Export instructions
7. Website integration notes
