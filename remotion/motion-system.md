# Motion System

## Composition types

### ProjectHeroLoop

Purpose:
A short looping hero asset for a project page.

Recommended:
- 6–10 seconds
- 1920x1080 or project-specific ratio
- subtle movement
- no audio unless requested

### CaseStudyReel

Purpose:
A short sequence showing highlights of the project.

Recommended:
- 10–15 seconds
- image sequence
- typography overlays
- premium transitions

### KineticTypeReveal

Purpose:
A typography-led motion moment for project title, campaign statement, or project concept.

Recommended:
- 3–6 seconds
- restrained
- brand type only

### ProcessBreakdown

Purpose:
Shows stages, layers, systems, or transformations.

Recommended:
- 8–12 seconds
- clear step rhythm
- useful for explaining design craft

## Easing style

Use smooth, calm easing.

Avoid:
- elastic
- bounce
- extreme overshoot
- fast chaotic motion

## Integration

Website should use:
- poster image
- lazy loading
- compressed video
- reduced-motion fallback
