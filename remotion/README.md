# Remotion Setup

## Purpose

This folder is for portfolio motion graphics.

Use Remotion for:

- project hero loops
- animated case-study assets
- typography reveals
- mockup animations
- process breakdowns
- social preview videos

## Install

From the repo root or inside this folder:

```bash
npx create-video@latest
```

## Recommended Structure

```text
remotion/
├─ src/
│  ├─ Root.tsx
│  ├─ compositions/
│  │  ├─ ProjectHeroLoop.tsx
│  │  ├─ CaseStudyReel.tsx
│  │  └─ KineticTypeReveal.tsx
│  ├─ components/
│  └─ data/
└─ renders/
```
