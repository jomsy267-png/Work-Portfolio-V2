# Typography Rules

## Purpose

Typography should make the portfolio feel premium, not busy.

## Rules

- Use existing fonts.
- Keep line lengths comfortable.
- Use large type only when it has a clear role.
- Avoid too many weights.
- Avoid too many sizes.
- Use labels and metadata with restraint.
- Do not over-style body copy.
- Let imagery and spacing carry the page.

## Project page type hierarchy

Recommended hierarchy:

1. Project title
2. One-line positioning statement
3. Project metadata
4. Section title
5. Short supporting body copy
6. Captions only where useful

## Case-study writing tone

Write like a premium design studio:
- clear
- confident
- precise
- human
- not corporate
- not over-explained
- not full of buzzwords
