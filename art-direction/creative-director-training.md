# Creative Director Training System

## Purpose

This file teaches Codex how to think before designing.

Codex cannot literally self-train. Instead, it must build reusable creative judgment by auditing, applying, critiquing, and documenting decisions.

## The art director loop

For every project:

1. Understand the brand.
2. Understand the project.
3. Find the strongest visual story.
4. Remove weak noise.
5. Create hierarchy.
6. Sequence images intentionally.
7. Add motion only where meaningful.
8. Review against agency-level standards.
9. Save learnings.

## Questions before redesigning

Codex must answer these before coding:

1. What is the strongest asset in this project?
2. What is the weakest part of the current page?
3. What should the first viewport communicate?
4. What should be the hero image?
5. What should be full-bleed?
6. What should be shown as details?
7. What should be removed?
8. Where should the page slow down?
9. Where should the page create impact?
10. What motion would improve the story?

## Top-end art direction standards

A strong project page should have:

- a clear point of view
- confident spacing
- excellent image hierarchy
- visual rhythm
- strong first impression
- controlled typography
- clear project context
- one or two memorable moments
- consistent brand behaviour
- no filler sections

## What a top art director does

A top art director does not ask:
"How can I add more?"

They ask:
"What is the clearest, sharpest, most memorable way to show this work?"

## The agency filter

Before finalizing any project page, Codex must ask:

- Would this feel good beside a premium studio case study?
- Does it look custom or templated?
- Is the best work shown early enough?
- Are the images sequenced with intention?
- Is there enough quiet space?
- Is anything trying too hard?
- Did we protect the brand?
