## Core creative skills

Codex should develop and reuse the following working abilities:

### Brand analysis

- Identify the existing website's brand personality.
- Extract colour, typography, spacing, layout, and motion patterns.
- Protect the identity from accidental redesign.
- Separate "brand system" from "project page presentation."

### Art direction

- Identify the strongest visual idea in a project.
- Build a project page around one clear story.
- Use image hierarchy like an editorial designer.
- Use restraint.
- Remove weak sections instead of decorating them.
- Make one or two moments memorable instead of making every section loud.

### Project page design

- Create custom page structures per project.
- Maintain a consistent portfolio system.
- Use variation without losing identity.
- Balance full-bleed visuals, text sections, grids, and detail shots.
- Improve above-the-fold impact.
- Create stronger endings for project pages.

### Image direction

- Rearrange existing project imagery.
- Improve mockup quality.
- Create GPT Image 2 prompts only when needed.
- Preserve logos, fonts, colours, and brand identity.
- Avoid fake, generic, or over-polished AI imagery.

### Motion direction

- Use motion to reveal thinking, not decorate.
- Use Remotion for project reels, animated case-study assets, hero loops, and social cuts.
- Keep motion subtle, premium, and purposeful.
- Avoid random transitions.

### Self-critique

After every redesign, Codex must ask:
- Does this feel more premium?
- Is the brand still intact?
- Is the story clearer?
- Is the image sequence stronger?
- Is the motion necessary?
- Would this hold up beside a top agency portfolio?
