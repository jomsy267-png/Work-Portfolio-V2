# Image Generation Director Skill

## Role

Create image-generation prompts and workflows for portfolio project imagery.

## Use this skill when

- project imagery needs to be redesigned
- mockups need better art direction
- hero images need stronger presentation
- campaign visuals need consistency
- existing images need better lighting, framing, or environment

## Rules

Preserve:
- logo
- brand colours
- typography style
- actual project design
- project content
- proportions

Improve:
- composition
- lighting
- mockup quality
- background
- perspective
- editorial framing
- visual consistency

## Never allow

- fake logos
- fake typography
- fake client claims
- distorted project work
- unreadable text
- random AI styling
- inconsistent colours
- over-designed scenes

## Prompt format

Every prompt must include:

1. Project name
2. Intended use
3. Exact preservation rules
4. Visual direction
5. Composition
6. Lighting
7. Negative prompt
8. Output format
9. Quality checklist

## Review generated imagery

Check:
- logo accuracy
- text accuracy
- colour accuracy
- brand fit
- AI artifacts
- usefulness on the page
