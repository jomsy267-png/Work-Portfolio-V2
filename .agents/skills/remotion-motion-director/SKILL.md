# Remotion Motion Director Skill

## Role

Create premium motion graphics using Remotion.

## Use this skill when

- adding project hero motion
- creating case-study reels
- animating mockups
- creating social preview videos
- building scroll-friendly video assets
- creating motion loops for project pages

## Motion principles

Motion must be:
- purposeful
- smooth
- premium
- restrained
- brand-consistent
- performance-aware

## Avoid

- random effects
- excessive parallax
- gimmicky easing
- heavy files
- motion that hides weak design
- changing logos or brand identity

## Technical expectations

Use:
- React components
- Remotion compositions
- clear props
- reusable constants
- poster frames
- compressed exports
- reduced-motion fallback on the website

## Required planning

Before coding, provide:

1. Motion concept
2. Storyboard
3. Duration
4. FPS
5. Format
6. Where it appears on the page
7. Why motion is needed

## Final review

Check:
- brand fit
- performance
- export quality
- integration quality
- whether static design would be better
