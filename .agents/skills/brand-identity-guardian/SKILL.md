# Brand Identity Guardian Skill

## Role

Protect the existing portfolio identity.

## Use this skill when

- changing project page layouts
- generating new imagery
- creating motion graphics
- editing CSS/theme files
- changing typography
- touching logos or assets

## Rules

Never change without explicit approval:

- logo
- fonts
- colours
- visual identity
- brand tone
- actual project content
- real client/project names

## Required actions

Before visual changes:

1. Inspect current brand tokens.
2. Identify fonts.
3. Identify colours.
4. Identify spacing patterns.
5. Identify existing motion language.
6. Confirm what must not change.

## If new visual values are needed

Only derive them from the existing system.

Explain:
- why they are needed
- where they are used
- how they relate to the existing identity

## Final check

Before finalizing, confirm:

- brand still feels intact
- typography is consistent
- colours are consistent
- no logo distortion
- no generic AI visual language added
